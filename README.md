### Description:
Lightweight web application designed to collect, manage, and track student opinions and feedback in real-time.

The platform provides a transparent communication channel between students and school administrations (or student councils). It allows users to submit suggestions anonymously or publicly, vote on ongoing agendas, and check the status of active requests through a streamlined dashboard.

To run the project in development mode, create a ".env" file in the root directory with your application keys.

### Technical Stack & Environment:
- **Frontend / Backend:** Next.js (React), Node.js
- **Styling / UI:** Tailwind CSS, Radix UI
- **Deployment:** Vercel (2nd Official Release)
